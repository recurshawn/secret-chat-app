# Technology Stack

## Frontend
- **Framework:** [Next.js](https://nextjs.org/) (v15.1.6) using the App Router.
- **Library:** [React](https://react.dev/) (v19.0.0).
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) (v4.0.0) for utility-first styling.
- **Language:** [TypeScript](https://www.typescriptlang.org/) for type-safe development.

## Communication & Persistence
- **Real-time:** [Socket.io](https://socket.io/) for WebSocket-based bidirectional communication.
- **Local Storage:** Browser [Web Storage API (LocalStorage)](https://developer.mozilla.org/en-US/docs/Web/API/Window/localStorage) for client-side message persistence and "self-destruct" functionality.

## Development Tools
- **Linter:** [ESLint](https://eslint.org/) for code quality.
- **Environment:** Node.js runtime.
